# Data Structures & Algorithms Learning Syllabus

## 📋 Overview
A structured learning program covering Data Structures and Algorithms fundamentals. This syllabus provides a comprehensive pathway from basic concepts to advanced problem-solving techniques.

## 🎯 Learning Objectives
By completing this syllabus, you will be able to:
- Understand fundamental data structures (arrays, linked lists, trees, graphs, etc.)
- Apply core algorithmic techniques and design patterns
- Solve complex coding problems efficiently
- Prepare effectively for technical interviews

## 📖 How to Study
1. **Read** - Understand foundational concepts first
2. **Learn** - Progress through modules in sequential order
3. **Practice** - Each lesson includes problem statements, solutions, code, and test cases
4. **Apply** - Modify and experiment with solutions

## 📝 Topics Covered
- Arrays and Strings
- Linked Lists
- Stacks and Queues
- Trees (Binary Trees, BST, AVL Trees)
- Graphs (BFS, DFS, Dijkstra, Bellman-Ford)
- Hash Tables and HashMaps
- Sorting Algorithms (Bubble, Merge, Quick, Heap)
- Searching Algorithms (Linear, Binary)
- Dynamic Programming
- Greedy Algorithms
- Two Pointer Technique
- Sliding Window
- Recursion and Backtracking
- Bit Manipulation
- Math and Number Theory
- String Matching and Pattern Searching

## ✅ Study Tips
- Write code by hand first before typing
- Understand the "why" behind each solution
- Practice multiple variations of similar problems
- Time yourself on problem-solving exercises
- Revisit difficult problems after a few days
